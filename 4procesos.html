SIREC D200

Es una grabadora de pantalla competitiva en formato de 144 mm x 144 mm con pantalla LCD a color de 5,7" en resolución QVGA.

• Tasa de muestreo 100/200/500 ms
• Precisión: 0,1% típico -T/C
• Fuente de alimentación del transmisor 24Vdc
• Funciones matemáticas
• Búfer de datos interno de 1 GB (2 GB opcional)
• Interfaz: Ethernet TCP/IP
• Registro de datos a través de interfaz USB o tarjeta SD
• Configuración en el dispositivo o mediante SIREC D Manager o Server-Software
• Pantalla táctil con navegación rápida
• Hasta 4 niveles de protección de contraseña con hasta 50 usuarios diferentes disponibles.
• Protección frontal tipo NEMA 3/IP54, NEMA 4X/IP66 disponible como opción
• Configuración en el dispositivo o mediante SIREC D Manager o Server-Software
• Opciones de firmware seleccionables individualmente por un sistema de crédito flexible

SIREC D300

Es un grabador de pantalla multifuncional en formato de 144 mm x 144 mm con pantalla LCD a color de 5,7" en resolución VGA.

• Tasa de escaneo rápido de 20/100/200/500 ms
• De 1 GB a 4 GB de búfer de datos interno
• Precisión: 0,1% típico -T/C
• Fuente de alimentación del transmisor 24Vdc
• Funciones matemáticas integrales
• Interfaces: Ethernet, RS 485, USB, ranura para tarjeta SD
• Registro de datos a través de interfaz USB o tarjeta SD
• Operación rápida a través de una pantalla táctil
• Configuración en el dispositivo o a través del software SIREC D manager o server
• Hasta 4 niveles de protección de contraseña con hasta 50 usuarios diferentes disponibles
• Protección frontal tipo NEMA 3/IP65, NEMA 4X/IP66 disponible como opción
• Opciones de firmware seleccionables individualmente a través de un sistema de crédito flexible
